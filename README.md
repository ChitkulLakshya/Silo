# Silo

Silo is a clean, isolated workspace application designed for managing work emails, Google Drive, and Google Photos without personal account crossover. It ensures absolute data boundary separation between work and personal profiles.

## Architecture

- **Isolation Engine**: Standardized security boundaries preventing cross-account token leakage.
- **Mail Integration**: Secure workspace email client built on top of Gmail API.
- **File Explorer**: Isolated access to Google Drive work repositories.
- **Photos Gallery**: Clean, secure interface for managing business and product visual assets.

## Setup & Running

Instructions on how to configure and run the application locally can be found inside the subdirectories (e.g., `zen-mail`).
